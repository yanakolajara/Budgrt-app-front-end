# Budgrt Application

Budgtr is an application where people can log their financial transactions. 

# Setup intructions

- Clone this repository
`git clone https://github.com/yanakolajara/budgrt-app-front-end`

- Install npm
`npm install`

- Install libraries
`npm i react router axios`

- Start live server
`npm start`
